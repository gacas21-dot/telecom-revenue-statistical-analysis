# telecom-revenue-statistical-analysis
nálisis de rentabilidad para Megaline mediante estadística inferencial. Procesé perfiles de consumo de 500 usuarios, aplicando pruebas de hipótesis (t-student) con SciPy para comparar los ingresos de los planes Surf y Ultimate. Identifiqué patrones de exceso de datos clave para optimizar la estrategia de marketing y el presupuesto.
